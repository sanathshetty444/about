Live Video Streaming Platform

TypeScript React Node.js WebRTC MediaSoup

A scalable video streaming application built with MediaSoup SFU architecture, supporting real-time multi-participant video conferences with optimal bandwidth usage. This platform allows for seamless audio and video streaming with features like screen sharing, chat functionality, and room management.

The architecture employs a Selective Forwarding Unit (SFU) model that optimizes bandwidth by intelligently routing media streams between participants, making it suitable for large meetings with limited network resources.

Anti-Cheat Online Invigilator

JavaScript React Node.js TensorFlow WebRTC

A proctoring system for online exams with advanced cheating detection mechanisms, ensuring academic integrity during remote assessments. This solution monitors students during online exams using real-time video analysis, eye-tracking, and browser activity monitoring.

The system uses machine learning algorithms to detect suspicious behaviors such as looking away from the screen, the presence of additional people, or the use of unauthorized materials. All detections generate alerts for human proctors to review.

90% cheating detection accuracy

10,000+ proctored exams
